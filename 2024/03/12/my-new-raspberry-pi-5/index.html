繁嵐的部落格

樹莓派第五代開箱

2024-03-12
最後修改於 2024-03-13

在先前購買了第三代改款樹莓派，並且先後作為家裡的影音娛樂機上盒以及伺服器使用之後，對於這樣的能耗比非常地滿意。後來第四代推出，就遇到了全世界的疫情，讓大家對於樹莓派的需求大增，因此價格高漲還缺貨，在這樣的情況下，考慮到只相差一代，我也沒有迫切迭代的需求，因此就等到了今年推出第五代，同時供貨狀況與價格也趨於平穩，我就馬上填了預購單，算是成為了臺灣首批入手 Raspberry Pi 5 的人。

與我手上第三代的比較

跟我現有的第三代相比，光是在外觀介面上就又有了許多改版，像是從供電口從 Mirco USB 改為 Type-C、一個全尺寸 HDMI 改為兩個 Micro HDMI，移除耳機孔和增加開機鈕等等，而內部規格當然是也有了大幅度的提升，可以更符合現代的使用需求。

簡易比較表：

Raspberry Pi 3 b+ Raspberry Pi 5
CPU 1.4GHz 1.8GHz
GPU VideoCore IV (400 MHz) VideoCore VII (800 MHz)
RAM 1GB LPDDR2 4GB LPDDR4x
USB 4xUSB2 2xUSB2、2xUSB3
視訊輸出 1xHDMI 2xMicro HDMI
耳機孔 O X
開機鈕 X O
本體價格 ~1500 ~2200

正式開箱

現今的樹莓派包裝都是盡量簡約，彩色印刷的紙盒剛好把樹莓派本體包裹其中，另外就是幾張說明書與注意事項。

而這一代的機殼因應峰值功耗上升，官方機殼主動加裝了風扇替樹莓派散熱，也算是跟前幾代官方機殼最大的不同。

後記

這次購買樹莓派，主要是用來當作租屋處的智慧家庭整合平臺，因為各類智慧家居產品有些只支援 Apple HomeKit，有些則是支援 Google Home，所以就透過 Home Assistant 這套解決方案來統一裝置間的溝通與自動化。

在使用約兩週後，對它安靜高效的表現很滿意，完全符合我的需求。如果是有高負載需求的人，可能要考慮其他更積極的主動式散熱套組，以避免過熱產生降頻。
